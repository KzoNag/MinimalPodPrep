# Minimal Pod Prep

ブラウザだけで動作する単一ファイルのポッドキャスト編集ツールです。会話音声のトリミング、BGMのループ合成とダッキング、プレビュー再生、WAV/MP3書き出しに対応しています。

## 主な機能
- 会話音声をアップロードし、波形ハンドルでトリミング。
- BGMをアップロードし、会話中の音量（ダッキング）を調整。
- シークバーで任意位置からプレビュー再生。
- WAV/MP3で書き出し（MP3はCDN読み込みのエンコーダを使用）。

## 使い方
1. ブラウザで `index.html` を開きます。
2. 会話音声をアップロードし、トリミング範囲を調整します。
3. BGMをアップロードし、必要に応じてダッキングを調整します。
4. シークバーとプレビュー再生でタイミングを確認します。
5. WAVまたはMP3で書き出します。

## 注意点
- MP3書き出しは `lamejs` をCDNから読み込んでいます。オフライン環境ではWAV書き出しを使うか、エンコーダをローカル同梱してください。
- 書き出し時間が長い場合があります。進捗バーはレンダリングとエンコード中に更新されます。
